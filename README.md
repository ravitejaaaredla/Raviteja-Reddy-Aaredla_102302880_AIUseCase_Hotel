# Hotel Booking Chatbot

## Project Overview

This project is a web-based hotel booking chatbot developed using Python and Flask.
The chatbot assists users in booking a hotel room through a straightforward chat interface.
It collects user details, including name, email, check-in date, check-out date, number of guests, room type, and breakfast preference.

## Features

* Conversational hotel room booking
* Rule-based chatbot logic
* Input validation for email, dates, and guests
* FAQ support for common hotel questions
* Session-based chat flow
* Booking summary generation
* Simple and responsive user interface

## Technologies Used

* Python
* Flask
* HTML
* CSS
* Jinja2
* Session storage

## Project Structure

* hotel_chatbot.py – main chatbot backend
* index.html – chatbot user interface
* style.css – styling for the interface
* requirements.txt – required Python packages
* booking_*.json – sample booking data files
